This MATLAB FileExchange entry contains files related to the "Object Detection Using Blob Analysis" video of the AUVSI Foundation: Computer Vision Training.

examples - Files used in the video
[~] blob_start.m : Example starter MATLAB code file to detect objects in images using blob analysis.
[~] blobDetection.m : Example MATLAB code file to detect objects in images using blob analysis.
[~] allFramesBlob.m : Example MATLAB code file to detect objects in videos using blob analysis.
[~] sampleFrames.mat : Example MAT-file with video frames showing green buoys.
[~] buoyRun.avi : Example video of a robot boat navigating a field of red and green buoys.

exercises - Files for exercise and practice tasks
[~] auvsi_ipcv_exercises_02_note.pdf : Note introducing the exercise tasks and presenting respective solutions
[~] thresholdSign.m : Solution MATLAB code file for the exercise "Thresholding a Traffic Sign"
[~] extractBlobSign.m : Solution MATLAB code file for the exercise "Blob Analysis for a Traffic Sign"
[~] thresholdImage.m : Helper MATLAB function which thresholds the given image to extract the red background of a 		                       traffic sign
[~] frame.mat : MAT-file with a video frame showing a stop sign
